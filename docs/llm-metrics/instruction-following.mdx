# Instruction Following Metric

You gave the LLM a list of rules. Did it follow them? This metric checks whether responses actually adhere to specific instructions — format requirements, style guidelines, content constraints, whatever you've specified.

## Quick Example

```python
import fasteval as fe

@fe.instruction_following(
    instructions=["Be concise", "Use bullet points", "Include examples"],
    threshold=0.8
)
def test_format_compliance():
    response = my_llm("Explain the benefits of unit testing")
    fe.score(response, input="Explain the benefits of unit testing")
```

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `instructions` | list[str] | Required | List of instructions to evaluate against |
| `threshold` | float | 0.8 | Minimum score to pass (0.0 - 1.0) |
| `model` | str | None | Override the default LLM model |

## How Scoring Works

The score is calculated as: `instructions_followed / total_instructions`

| Score | Interpretation | Example |
|-------|---------------|---------|
| **1.0** | All instructions followed | 5/5 instructions met |
| **0.8** | Most instructions followed | 4/5 instructions met |
| **0.6** | Some instructions followed | 3/5 instructions met |
| **0.0** | No instructions followed | 0/5 instructions met |

For each instruction, the LLM judge checks whether it's clearly followed, partially complied with, or violated outright.

## Practical Examples

### Format Requirements

```python
@fe.instruction_following(
    instructions=[
        "Response must be under 100 words",
        "Use numbered list format",
        "Include a summary at the end"
    ],
    threshold=1.0  # All instructions must be followed
)
def test_strict_format():
    response = my_llm("List the top 3 programming languages for beginners")
    fe.score(response)
```

### Style Guidelines

```python
@fe.instruction_following(
    instructions=[
        "Use professional tone",
        "Avoid jargon",
        "Address the reader as 'you'"
    ],
    threshold=0.8
)
def test_style():
    response = my_llm("Explain how to open a bank account")
    fe.score(response)
```

### Content Constraints

```python
@fe.instruction_following(
    instructions=[
        "Do not mention competitor products",
        "Include a call to action",
        "Mention our 30-day guarantee"
    ],
    threshold=1.0
)
def test_marketing_copy():
    response = my_llm("Write a product description for our cloud storage service")
    fe.score(response)
```

### Multi-Constraint Prompts

```python
@fe.instruction_following(
    instructions=[
        "Start with a question",
        "Include exactly 3 main points",
        "End with a recommendation",
        "Keep total length under 200 words"
    ],
    threshold=0.75  # Allow some flexibility
)
def test_blog_intro():
    response = my_llm("Write a blog intro about AI in education")
    fe.score(response)
```


## Choosing Your Threshold

Soft guidelines (nice-to-haves) work at 0.6-0.7. Standard compliance should sit at 0.75-0.85. Strict format requirements need 0.9-1.0. And for critical compliance — legal or regulatory — set it to 1.0. No exceptions.

## When to Use Instruction Following

This metric shines for output format validation, style guide compliance, content policy enforcement, template adherence, and multi-constraint prompts. If you're checking content accuracy, use `@fe.correctness()`. For overall quality, combine `@fe.coherence()` with `@fe.helpfulness()`. For single pattern matching, `@fe.regex()` is simpler and faster.

## Combining with Other Metrics

For content generation with quality requirements:

```python
@fe.instruction_following(
    instructions=["Use formal tone", "Include citations"],
    threshold=0.9
)
@fe.correctness(threshold=0.8)
@fe.coherence(threshold=0.8)
def test_academic_content():
    response = my_llm("Explain climate change causes")
    fe.score(response, "Human activities like burning fossil fuels...")
```

For customer-facing content:

```python
@fe.instruction_following(
    instructions=[
        "Be empathetic",
        "Provide clear next steps",
        "Include support contact info"
    ],
    threshold=1.0
)
@fe.helpfulness(threshold=0.8)
@fe.toxicity(threshold=0.95)
def test_support_response():
    response = my_llm("Customer complaint: My order hasn't arrived")
    fe.score(response)
```

In practice, instruction following pairs well with almost any other metric. It handles the "how" (format, style, constraints) while other metrics handle the "what" (accuracy, safety, usefulness).
