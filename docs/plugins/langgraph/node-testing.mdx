# Node Testing

You don't always need to run the full graph. `.node()` lets you execute a single node in isolation, inspect its state updates, and verify its routing decisions.

## Single Node Execution

Call `.node(name).run()` with the input state you want:

```python
from fasteval_langgraph import harness
from langchain_core.messages import HumanMessage

graph = harness(compiled_graph)

result = await graph.node("classifier").run(
    messages=[HumanMessage(content="My app keeps crashing")]
)
```

### NodeResult Fields

| Field | Type | Description |
|-------|------|-------------|
| `node_name` | `str` | Name of the executed node |
| `updates` | `dict` | State updates written by the node |
| `goto` | `str \| None` | Routing target from `Command`, if any |
| `response` | `str \| None` | AI message content, if any |
| `execution_time_ms` | `float` | Execution time in milliseconds |

## Testing Command Routing

Nodes that return `Command(goto=..., update=...)` have their routing parsed automatically:

```python
result = await graph.node("classifier").run(
    messages=[HumanMessage(content="I need to troubleshoot this")]
)
assert result.goto == "planner"
assert result.updates["intent"] == "TROUBLESHOOTING"

result2 = await graph.node("classifier").run(
    messages=[HumanMessage(content="What is OAuth?")]
)
assert result2.goto == "rag"
assert result2.updates["intent"] == "FAQ"
```

## Combining with Assertions

Mix hard assertions with fasteval's LLM evaluation in the same test:

```python
import fasteval as fe

@fe.criteria("Does the classifier correctly identify troubleshooting queries?")
@pytest.mark.asyncio
async def test_classifier():
    result = await graph.node("classifier").run(
        messages=[HumanMessage(content="My app crashes on startup")]
    )
    # Deterministic assertions
    assert result.goto == "planner"
    assert result.updates["intent"]["category"] == "TROUBLESHOOTING"
    assert result.execution_time_ms < 5000

    # LLM evaluation
    fe.score(
        str(result.updates["intent"]),
        "TROUBLESHOOTING classification for app crash",
        input="My app crashes on startup",
    )
```

## Trace Inspection

When you use `.chat()`, the trace captures every node that executed:

```python
result = await graph.chat("My API returns 401")

for step in result.trace:
    print(f"{step.node}: {step.updates}")

# Example output:
# classifier: {"intent": "TROUBLESHOOTING"}
# rag: {"docs": ["auth troubleshooting guide..."]}
# planner: {"plan": {"paths": [...]}}
# responder: {"messages": [...]}
```

### Assert on Trace

```python
# Verify the correct execution path
assert result.nodes_ran == ["classifier", "rag", "planner", "responder"]

# Verify a specific node produced expected updates
rag_step = next(s for s in result.trace if s.node == "rag")
assert len(rag_step.updates.get("docs", [])) > 0
```

### Graph Inspection

```python
graph = harness(compiled_graph)

graph.nodes              # {"classifier", "rag", "planner", "responder"}
graph.entry              # "classifier"
graph.has_messages_state # True
```
