# Examples

This page provides complete, copy-paste ready examples for common evaluation workflows. Each example includes the full test code, expected output, and guidance on when to use that pattern.

All examples assume you have:
- Langfuse credentials configured (via environment variables or `configure_langfuse`)
- Production traces or datasets already in your Langfuse project
- fasteval and fasteval-langfuse installed (`pip install fasteval fasteval-langfuse`)

## Production Monitoring

Monitor production quality continuously with hourly evaluations:

```python
# test_production_monitoring.py
from fasteval_langfuse import langfuse_traces
from fasteval_langfuse.sampling import RandomSamplingStrategy
import fasteval as fe

@fe.correctness(threshold=0.8)
@fe.hallucination(threshold=0.9)
@fe.toxicity(threshold=0.95)
@langfuse_traces(
    project="production",
    filter_tags=["customer-facing"],
    time_range="last_1h",
    sampling=RandomSamplingStrategy(sample_size=100)
)
def test_hourly_quality_check(trace_id, input, output, context, metadata):
    """Evaluate 100 random traces from the last hour."""
    expected = metadata.get("expected_answer")
    fe.score(output, expected, input=input)
```

**Schedule with cron:**

```bash
# Run every hour
0 * * * * cd /app && source .venv/bin/activate && pytest test_production_monitoring.py

# With logging
0 * * * * cd /app && pytest test_production_monitoring.py >> /var/log/evaluations.log 2>&1

# With alerting
0 * * * * cd /app && pytest test_production_monitoring.py || send_alert "Evaluation failed"
```

**Expected Output:**

```
Evaluating 100/1,243 traces (8.0% sample, strategy=RandomSamplingStrategy)

test_hourly_quality_check PASSED

Evaluation Results:
  ✓ correctness: 0.87 (threshold: 0.80)
  ✓ hallucination: 0.94 (threshold: 0.90)
  ✓ toxicity: 0.98 (threshold: 0.95)
  Aggregate: 0.93

Scores pushed to 100 traces in Langfuse ✅
```

## Debug Failing Traces

Focus evaluation budget on problematic interactions:

```python
# test_debug_failures.py
from fasteval_langfuse.sampling import ScoreBasedSamplingStrategy

@fe.relevance(threshold=0.7)
@fe.faithfulness(threshold=0.8)
@fe.contextual_precision(threshold=0.7)
@langfuse_traces(
    project="production",
    filter_tags=["rag", "customer-support"],
    time_range="last_7d",
    sampling=ScoreBasedSamplingStrategy(
        score_name="user_rating",
        low_score_threshold=3.0,
        low_score_rate=1.0,      # 100% of ratings ≤ 3.0
        high_score_rate=0.05,    # 5% of ratings > 3.0
    )
)
def test_low_rated_interactions(trace_id, input, output, context, metadata):
    """Focus on low-rated interactions to identify root causes."""
    # Context auto-extracted
    if not context:
        context = metadata.get("retrieved_docs")
    
    expected = metadata.get("expected_answer")
    fe.score(output, expected, context=context, input=input)
    
    # Debug logging
    user_rating = next(
        (s["value"] for s in metadata.get("scores", []) if s["name"] == "user_rating"),
        None
    )
    print(f"Trace {trace_id}: user_rating={user_rating}")
```

**Analysis:**

```bash
pytest test_debug_failures.py -v -s  # -s shows print output
```

**Identifies issues:**
- Low `contextual_precision` → Retrieval problem
- Low `faithfulness` → Hallucination problem
- Low `relevance` → Understanding problem

## A/B Testing

Compare prompt versions, models, or configurations:

```python
# test_ab_comparison.py

# Version A: New prompt with chain-of-thought
from fasteval_langfuse import langfuse_dataset

@fe.correctness(threshold=0.85)
@fe.relevance(threshold=0.8)
@langfuse_dataset(name="qa-golden-set", version="v3")
def test_cot_prompt(input, expected_output):
    """Test chain-of-thought prompt."""
    response = agent_with_cot_prompt(input)
    fe.score(response, expected_output, input=input)


# Version B: Baseline direct prompt
@fe.correctness(threshold=0.85)
@fe.relevance(threshold=0.8)
@langfuse_dataset(name="qa-golden-set", version="v3")
def test_direct_prompt(input, expected_output):
    """Test direct prompt (baseline)."""
    response = agent_with_direct_prompt(input)
    fe.score(response, expected_output, input=input)


# Helper functions
def agent_with_cot_prompt(query: str) -> str:
    """Agent using chain-of-thought prompting."""
    cot_template = """
    Think step by step to answer this question:
    {query}
    
    Let's approach this systematically:
    """
    return llm.invoke(cot_template.format(query=query))


def agent_with_direct_prompt(query: str) -> str:
    """Agent using direct prompting."""
    return llm.invoke(query)
```

**Run comparison:**

```bash
pytest test_ab_comparison.py -v --html=ab_results.html
```

**Results analysis:**

| Test | Correctness | Relevance | Aggregate |
|------|-------------|-----------|-----------|
| test_cot_prompt | 0.91 ✅ | 0.85 ✅ | 0.88 |
| test_direct_prompt | 0.87 ✅ | 0.82 ✅ | 0.845 |

**Winner**: Chain-of-thought (+4% improvement)

## RAG Pipeline Evaluation

Comprehensive RAG evaluation with context:

```python
# test_rag_pipeline.py
from fasteval_langfuse import langfuse_traces
from fasteval_langfuse.sampling import RandomSamplingStrategy

@fe.faithfulness(threshold=0.85)
@fe.contextual_precision(threshold=0.75)
@fe.contextual_recall(threshold=0.75)
@fe.answer_correctness(threshold=0.8)
@langfuse_traces(
    project="production",
    filter_tags=["rag", "product-docs"],
    time_range="last_24h",
    sampling=RandomSamplingStrategy(sample_size=200)
)
def test_rag_quality(trace_id, input, output, context, metadata):
    """Evaluate RAG pipeline quality."""
    # Context auto-extracted from metadata
    if not context:
        print(f"WARNING: No context for trace {trace_id}")
        return
    
    # Get ground truth if available
    expected = metadata.get("expected_answer")
    
    # Evaluate all RAG metrics
    fe.score(output, expected, context=context, input=input)
```

**Example trace metadata:**

```json
{
  "retrieved_docs": [
    "Password reset instructions: Navigate to Settings...",
    "Security best practices: Use strong passwords..."
  ],
  "retrieval_score": 0.92,
  "num_docs_retrieved": 2,
  "expected_answer": "Go to Settings > Security > Reset Password"
}
```

**Run and analyze:**

```bash
pytest test_rag_pipeline.py -v
```

## Cross-Segment Quality

Ensure fair quality across user segments:

```python
# test_segment_quality.py
from fasteval_langfuse.sampling import StratifiedSamplingStrategy

@fe.correctness(threshold=0.8)
@fe.relevance(threshold=0.75)
@fe.bias(threshold=0.9)
@langfuse_traces(
    project="production",
    filter_tags=["customer-query"],
    time_range="last_7d",
    sampling=StratifiedSamplingStrategy(
        strata_key="metadata.user_type",
        samples_per_stratum=50
    )
)
def test_quality_across_user_types(trace_id, input, output, context, metadata):
    """Ensure quality is fair across free/paid users."""
    user_type = metadata.get("user_type", "unknown")
    region = metadata.get("region", "unknown")
    
    # Evaluate
    result = fe.score(output, input=input)
    
    # Log for segment analysis
    print(f"user_type={user_type}, region={region}, score={result.aggregate_score:.2f}")
```

**Analysis output:**

```
user_type=free, region=us-west, score=0.87
user_type=paid, region=us-west, score=0.89
user_type=free, region=us-east, score=0.85
user_type=paid, region=us-east, score=0.88

Average by segment:
  free users: 0.86
  paid users: 0.885
  Difference: 2.9% (acceptable)
```

## Regression Testing

Test new agent version against production inputs:

```python
# test_regression.py

@fe.correctness(threshold=0.85)
@fe.relevance(threshold=0.8)
@langfuse_traces(
    project="production",
    filter_tags=["regression-test-set", "curated"],
    sampling=RandomSamplingStrategy(sample_size=300, seed=42)
)
def test_new_agent_version(trace_id, input, output, context, metadata):
    """
    Re-run new agent on production inputs.
    
    - 'output' is the OLD agent's response
    - We re-run with NEW agent and compare
    """
    # Re-run with new agent
    new_output = my_new_agent_v2(input, context=context)
    
    # Get expected from metadata or use old output as baseline
    expected = metadata.get("expected_answer") or output
    
    # Evaluate new version
    result = fe.score(new_output, expected, input=input, context=context)
    
    # Compare with old output score (if available in metadata)
    old_score = metadata.get("baseline_score")
    if old_score and result.aggregate_score < old_score - 0.05:
        print(f"REGRESSION: Score dropped from {old_score} to {result.aggregate_score}")


def my_new_agent_v2(query: str, context: list = None) -> str:
    """New agent version under test."""
    # Improved prompt, better model, etc.
    return llm_v2.invoke(query, context=context)
```

**Workflow:**

1. **Create regression test set** - Tag high-quality traces in Langfuse
2. **Run baseline** - Capture old agent scores
3. **Deploy new version** - Test with `test_new_agent_version()`
4. **Compare** - Ensure no regression, identify improvements

Evaluate conversation quality from production sessions:

## Custom Metric Testing

Use custom metrics with production traces:

```python
from fasteval import LLMMetric
from fasteval.core.models import EvalInput

class CustomerSatisfactionMetric(LLMMetric):
    """Custom metric for customer satisfaction."""
    
    def __init__(self, threshold: float = 0.8):
        super().__init__(
            name="customer_satisfaction",
            threshold=threshold
        )
    
    def get_evaluation_prompt(self, eval_input: EvalInput) -> str:
        return f"""
        Evaluate customer satisfaction for this support response.
        
        User Query: {eval_input.input}
        Agent Response: {eval_input.actual_output}
        
        Score from 0 to 1 based on:
        - Helpfulness
        - Professionalism
        - Completeness
        
        Respond with JSON: {{"score": <float>, "reasoning": "<explanation>"}}
        """


@fe.metric(CustomerSatisfactionMetric(threshold=0.8))
@langfuse_traces(project="production", filter_tags=["support"])
def test_custom_metric(trace_id, input, output, context, metadata):
    fe.score(output, input=input)
```

## Cost Analysis

Track evaluation costs across different strategies:

```python
# test_cost_analysis.py

# Scenario A: No sampling (expensive)
@fe.correctness(threshold=0.8)
@langfuse_traces(project="prod", time_range="last_1h")
def test_no_sampling(trace_id, input, output, context, metadata):
    fe.score(output, input=input)
# Cost: ~1,000 traces × $0.002 = $2.00/hour = $48/day


# Scenario B: Random sampling (cost-effective)
@fe.correctness(threshold=0.8)
@langfuse_traces(
    project="prod",
    time_range="last_1h",
    sampling=RandomSamplingStrategy(sample_size=100)
)
def test_with_sampling(trace_id, input, output, context, metadata):
    fe.score(output, input=input)
# Cost: ~100 traces × $0.002 = $0.20/hour = $4.80/day
# Savings: 90%


# Scenario C: Score-based sampling (maximum efficiency)
@fe.correctness(threshold=0.8)
@langfuse_traces(
    project="prod",
    time_range="last_1h",
    sampling=ScoreBasedSamplingStrategy(
        score_name="user_rating",
        low_score_threshold=3.0,
        low_score_rate=1.0,
        high_score_rate=0.05
    )
)
def test_score_based(trace_id, input, output, context, metadata):
    fe.score(output, input=input)
# Cost: ~60 traces × $0.002 = $0.12/hour = $2.88/day
# Savings: 94% while focusing on failures!
```

## Complete Workflow Example

End-to-end evaluation pipeline for a RAG-based customer support system:

```python
# test_customer_support_pipeline.py
from fasteval_langfuse import langfuse_traces, langfuse_dataset
from fasteval_langfuse.sampling import (
    RandomSamplingStrategy,
    ScoreBasedSamplingStrategy,
    StratifiedSamplingStrategy,
)
import fasteval as fe


# 1. Continuous Production Monitoring (Hourly)
@fe.correctness(threshold=0.8)
@fe.relevance(threshold=0.75)
@fe.toxicity(threshold=0.95)
@langfuse_traces(
    project="production",
    filter_tags=["customer-support"],
    time_range="last_1h",
    sampling=RandomSamplingStrategy(sample_size=100)
)
def test_hourly_monitoring(trace_id, input, output, context, metadata):
    """General quality check - random 100 traces."""
    fe.score(output, input=input)


# 2. Deep Dive on Failures (Daily)
@fe.faithfulness(threshold=0.8)
@fe.contextual_precision(threshold=0.7)
@fe.hallucination(threshold=0.9)
@langfuse_traces(
    project="production",
    filter_tags=["customer-support", "rag"],
    time_range="last_24h",
    sampling=ScoreBasedSamplingStrategy(
        score_name="user_rating",
        low_score_threshold=3.0,
        low_score_rate=1.0,
        high_score_rate=0.05,
    )
)
def test_daily_failure_analysis(trace_id, input, output, context, metadata):
    """Focus on failures - identify patterns."""
    if not context:
        context = metadata.get("retrieved_docs")
    
    expected = metadata.get("expected_answer")
    result = fe.score(output, expected, context=context, input=input)
    
    # Additional logging for root cause
    if result.aggregate_score < 0.7:
        print(f"FAILURE TRACE: {trace_id}")
        print(f"  Input: {input[:100]}")
        print(f"  Faithfulness: {result.metric_scores.get('faithfulness')}")
        print(f"  Contextual Precision: {result.metric_scores.get('contextual_precision')}")


# 3. Cross-Segment Fairness (Weekly)
@fe.correctness(threshold=0.8)
@fe.bias(threshold=0.9)
@langfuse_traces(
    project="production",
    filter_tags=["customer-support"],
    time_range="last_7d",
    sampling=StratifiedSamplingStrategy(
        strata_key="metadata.user_type",
        samples_per_stratum=50
    )
)
def test_weekly_fairness(trace_id, input, output, context, metadata):
    """Ensure quality across user segments."""
    user_type = metadata.get("user_type")
    result = fe.score(output, input=input)
    print(f"user_type={user_type}, score={result.aggregate_score:.2f}")


# 4. Regression Suite (Before Each Deploy)
@fe.correctness(threshold=0.9)
@fe.relevance(threshold=0.85)
@fe.faithfulness(threshold=0.85)
@langfuse_dataset(name="support-regression-suite", version="v2026-q1")
def test_pre_deploy_regression(input, expected_output, context):
    """Golden dataset - must pass before deploy."""
    response = customer_support_agent(input, context=context)
    fe.score(response, expected_output, context=context, input=input)
```

**Execution schedule:**

```bash
# Hourly monitoring
0 * * * * cd /app && pytest test_customer_support_pipeline.py::test_hourly_monitoring

# Daily failure analysis
0 2 * * * cd /app && pytest test_customer_support_pipeline.py::test_daily_failure_analysis

# Weekly fairness check
0 3 * * 0 cd /app && pytest test_customer_support_pipeline.py::test_weekly_fairness

# Pre-deploy (manual)
pytest test_customer_support_pipeline.py::test_pre_deploy_regression
```

## Parallel Execution

Speed up evaluation with pytest-xdist:

```bash
# Evaluate 1000 traces in parallel
pytest test_production.py -n 4  # 4 workers

# Auto-detect worker count
pytest test_production.py -n auto

# With verbose output
pytest test_production.py -n 4 -v
```

**Performance:**

```
Without parallelization: 1000 traces × 1.5s = 25 minutes
With 4 workers: 1000 traces / 4 × 1.5s = 6.25 minutes
Speedup: 4x
```

## CI/CD Integration

### GitHub Actions

```yaml
# .github/workflows/evaluate.yml
name: Production Evaluation

on:
  schedule:
    - cron: '0 */6 * * *'  # Every 6 hours

jobs:
  evaluate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      
      - name: Install dependencies
        run: |
          pip install fasteval fasteval-langfuse
      
      - name: Run evaluations
        env:
          LANGFUSE_PUBLIC_KEY: ${{ secrets.LANGFUSE_PUBLIC_KEY }}
          LANGFUSE_SECRET_KEY: ${{ secrets.LANGFUSE_SECRET_KEY }}
        run: |
          pytest test_production.py -v --html=report.html
      
      - name: Upload report
        uses: actions/upload-artifact@v3
        with:
          name: evaluation-report
          path: report.html
      
      - name: Alert on failure
        if: failure()
        uses: 8398a7/action-slack@v3
        with:
          status: ${{ job.status }}
          text: 'Production evaluation failed!'
        env:
          SLACK_WEBHOOK_URL: ${{ secrets.SLACK_WEBHOOK }}
```

## Best Practices

### 1. Start Simple

```python
# Begin with basic monitoring
@fe.correctness(threshold=0.8)
@langfuse_traces(
    project="prod",
    time_range="last_24h",
    sampling=RandomSamplingStrategy(sample_size=100)
)
def test_basic_monitoring(trace_id, input, output, context, metadata):
    fe.score(output, input=input)
```

### 2. Add Metrics Gradually

```python
# Week 1: Basic quality
@fe.correctness(threshold=0.8)

# Week 2: Add safety
@fe.correctness(threshold=0.8)
@fe.toxicity(threshold=0.95)

# Week 3: Add RAG metrics
@fe.correctness(threshold=0.8)
@fe.toxicity(threshold=0.95)
@fe.faithfulness(threshold=0.85)
```

### 3. Use Appropriate Sampling

- **Monitoring**: RandomSamplingStrategy (1-5%)
- **Debugging**: ScoreBasedSamplingStrategy (focus on failures)
- **Fairness**: StratifiedSamplingStrategy (across segments)

### 4. Monitor Trends

