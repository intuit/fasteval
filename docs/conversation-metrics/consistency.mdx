# Consistency Metric

Does your model contradict itself mid-conversation? This metric catches that. It evaluates whether the model maintains consistent information and behavior across turns.

## Usage

```python
import fasteval as fe

@fe.consistency(threshold=0.8)
@fe.conversation([...])
def test_consistent_answers(query, expected, history):
    response = my_agent(query, history=history)
    fe.score(response, expected, input=query, history=history)
```
