# Topic Drift Metric

Flags when a conversation wanders away from the original topic. In practice, we've found this especially useful for customer-support bots and task-oriented agents that shouldn't go off the rails.

## Usage

```python
import fasteval as fe

@fe.topic_drift(threshold=0.8)
@fe.conversation([...])
def test_stays_on_topic(query, expected, history):
    response = my_agent(query, history=history)
    fe.score(response, expected, input=query, history=history)
```
