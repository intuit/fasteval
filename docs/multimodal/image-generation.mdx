# Image Generation Metrics

Testing image generation models like DALL-E, Midjourney, or Stable Diffusion? fasteval gives you metrics for quality, prompt adherence, safety, aesthetics, and text-image alignment.

## Installation

```bash
# Basic vision support (LLM-based evaluation)
pip install fasteval-core[vision]

# Full image generation support (CLIP scores, etc.)
pip install fasteval-core[image-gen]
```

## Available Metrics

### `@fe.image_quality`

Evaluates visual quality of generated images — clarity, sharpness, freedom from artifacts, composition, and technical quality like lighting and perspective.

```python
import fasteval as fe

@fe.image_quality(threshold=0.7)
def test_generated_image():
    image = dalle.generate("A sunset over mountains")
    fe.score(
        generated_image=image,
        input="A sunset over mountains"
    )
```

**Parameters:**
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `threshold` | float | 0.7 | Minimum quality score to pass |
| `weight` | float | 1.0 | Weight for aggregate scoring |

### `@fe.prompt_adherence`

Checks whether the generated image actually matches what was asked for. It looks at whether all prompt elements are present, attributes are correct (colors, sizes, positions), and style matches the description.

```python
@fe.prompt_adherence(threshold=0.8)
def test_prompt_match():
    image = dalle.generate("A red sports car on a mountain road")
    fe.score(
        generated_image=image,
        input="A red sports car on a mountain road"
    )
```

### `@fe.safety_check`

Screens generated images for NSFW content, violence, hate symbols, dangerous activities, and content harmful to minors.

```python
@fe.safety_check(threshold=0.95)
def test_safe_generation():
    image = model.generate(prompt)
    fe.score(
        generated_image=image,
        input=prompt
    )
```

### `@fe.aesthetic_score`

Judges visual appeal and artistic merit — composition (balance, rule of thirds), color harmony, lighting quality, and creativity.

```python
@fe.aesthetic_score(threshold=0.7)
def test_image_aesthetics():
    image = model.generate("A beautiful sunset")
    fe.score(
        generated_image=image,
        input="A beautiful sunset"
    )
```

### `@fe.clip_score`

Measures image-text alignment using CLIP-style semantic similarity.

```python
@fe.clip_score(threshold=0.7)
def test_image_alignment():
    image = dalle.generate("A red sports car on a mountain road")
    fe.score(
        generated_image=image,
        input="A red sports car on a mountain road"
    )
```

**Note:** For actual CLIP embedding scores, install `fasteval-core[image-gen]` with PyTorch. The default uses LLM-based alignment evaluation.

## Combining Metrics

Stack multiple decorators for a thorough evaluation:

```python
@fe.image_quality(threshold=0.7)
@fe.prompt_adherence(threshold=0.8)
@fe.safety_check(threshold=0.95)
@fe.aesthetic_score(threshold=0.6)
def test_complete_generation():
    image = dalle.generate("A professional headshot photo")
    fe.score(
        generated_image=image,
        input="A professional headshot photo"
    )
```

## Working with Generated Images

### Using GeneratedImage Model

```python
from fasteval import GeneratedImage, ImageInput

# Track generation parameters
generated = GeneratedImage(
    image=ImageInput(source="output.png"),
    prompt="A red sports car on a mountain road",
    model="dall-e-3",
    seed=12345,
    generation_params={
        "quality": "hd",
        "style": "vivid"
    }
)

fe.score(generated_image=generated, input=prompt)
```

### Direct Image Path

```python
# Simple path-based usage
image_path = dalle.generate(prompt)  # Returns path
fe.score(generated_image=image_path, input=prompt)
```

## Example: A/B Testing Image Generation

```python
import fasteval as fe
from datetime import datetime

@fe.prompt_adherence(threshold=0.8)
@fe.aesthetic_score(threshold=0.7)
def test_model_comparison():
    prompt = "A cozy coffee shop interior with warm lighting"

    # Generate with model A
    image_a = model_a.generate(prompt)
    result_a = fe.score(
        generated_image=image_a,
        input=prompt,
        metadata={"model": "A"}
    )

    # Generate with model B
    image_b = model_b.generate(prompt)
    result_b = fe.score(
        generated_image=image_b,
        input=prompt,
        metadata={"model": "B"}
    )

    print(f"Model A: {result_a.aggregate_score:.2f}")
    print(f"Model B: {result_b.aggregate_score:.2f}")
```

## Best Practices

Always include `@fe.safety_check` for user-facing applications — it's non-negotiable. Keep in mind that a high aesthetic score doesn't mean the image followed the prompt well, so we recommend testing both dimensions separately. Include complex prompts with multiple elements in your test suite to catch edge cases. Use the `GeneratedImage` model to track seeds and settings so you can reproduce results. And set realistic thresholds — image generation is inherently variable, so don't expect perfection.
