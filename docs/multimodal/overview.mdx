# Multi-Modal Evaluation [Open in Google Colab](https://colab.research.google.com/github/intuit/fasteval/blob/main/docs/notebooks/multimodal-overview.ipynb)

AI doesn't just do text anymore. Vision-language models read images, Whisper transcribes audio, DALL-E generates images, and RAG pipelines pull from documents packed with tables and figures. fasteval is the first evaluation framework with native support across all of these.

## Installation

Multi-modal features are optional dependencies — install only what you need:

```bash
# Core (text-only) - no extra dependencies
pip install fasteval-core

# Vision-language evaluation (GPT-4V, Claude Vision)
pip install fasteval-core[vision]

# Audio/speech evaluation (Whisper, ASR)
pip install fasteval-core[audio]

# Image generation evaluation (DALL-E, Stable Diffusion)
pip install fasteval-core[image-gen]

# All multi-modal features
pip install fasteval-core[multimodal]

# Combine specific ones
pip install fasteval-core[vision,audio]
```

## Quick Examples

### Vision-Language Models

```python
import fasteval as fe

@fe.image_understanding(threshold=0.8)
def test_vision_qa():
    response = gpt4v.chat(
        image="chart.png",
        prompt="What trend does this chart show?"
    )
    fe.score(
        actual_output=response,
        expected_output="Revenue increased 25% year over year",
        image="chart.png"
    )
```

### Speech Recognition

```python
@fe.word_error_rate(threshold=0.95)  # WER < 5%
def test_whisper():
    transcript = whisper.transcribe("meeting.mp3")
    fe.score(
        actual_output=transcript,
        expected_output=ground_truth_transcript
    )
```

### Image Generation

```python
@fe.image_quality(threshold=0.7)
@fe.prompt_adherence(threshold=0.85)
def test_dalle():
    image = dalle.generate("A red sports car on a mountain road")
    fe.score(
        generated_image=image,
        input="A red sports car on a mountain road"
    )
```

### Multi-Modal RAG

```python
@fe.multimodal_faithfulness(threshold=0.8)
def test_document_qa():
    response = rag_system.query(
```

## Data Models

### ImageInput

Represents an image for evaluation:

```python
from fasteval import ImageInput

# From file path
image = ImageInput(source="path/to/image.png")

# From URL
image = ImageInput(source="https://example.com/image.jpg")

# From base64
image = ImageInput(source="data:image/png;base64,iVBOR...", format="base64")

# With metadata
image = ImageInput(
    source="chart.png",
    alt_text="Q3 Revenue Chart",
    metadata={"page": 5}
)
```

### AudioInput

Represents audio for evaluation:

```python
from fasteval import AudioInput

# From file path
audio = AudioInput(source="recording.mp3")

# With duration info
audio = AudioInput(
    source="speech.wav",
    duration_seconds=120.5,
    transcript="Optional ground truth transcript"
)
```

### GeneratedImage

For image generation evaluation:

```python
from fasteval import GeneratedImage, ImageInput

generated = GeneratedImage(
    image=ImageInput(source="output.png"),
    prompt="A red sports car on a mountain road",
    model="dall-e-3",
    seed=12345
)
```

## Extended `fe.score()` API

The `fe.score()` function accepts multi-modal parameters alongside the standard ones:

```python
fe.score(
    actual_output=response,
    expected_output="The chart shows 25% growth",
    input=query,
    context=docs,

    # Multi-modal parameters
    image="chart.png",                    # Single image
    images=["img1.png", "img2.png"],      # Multiple images
    audio="recording.mp3",                # Audio file
    generated_image=output_img,           # For image generation eval
    expected_fields={"total": "$1,234"},  # For OCR/extraction
)
```

## Metrics by Category

| Category | Metrics |
|----------|---------|
| **Vision-Language** | `image_understanding`, `ocr_accuracy`, `chart_interpretation`, `visual_grounding`, `image_faithfulness` |
| **Audio/Speech** | `word_error_rate`, `character_error_rate`, `transcription_accuracy`, `speaker_diarization`, `audio_sentiment` |
| **Image Generation** | `image_quality`, `prompt_adherence`, `safety_check`, `aesthetic_score`, `clip_score` |
| **Multi-Modal RAG** | `multimodal_faithfulness`, `table_extraction`, `figure_reference`, `cross_modal_coherence`, `document_understanding` |
