# Evaluator API Reference

The Evaluator runs metrics against evaluation inputs. You won't need it for most workflows — the decorator API handles things automatically — but it's there when you want direct control.

## Creating an Evaluator

```python
from fasteval import create_evaluator

evaluator = create_evaluator()
```

## Methods

### evaluate()

```python
async def evaluate(
    eval_input: EvalInput,
    metrics: List[MetricConfig]
) -> EvalResult
```

Runs the specified metrics on `eval_input` and returns an `EvalResult`.

## Example

```python
from fasteval import create_evaluator, EvalInput, MetricConfig

evaluator = create_evaluator()

result = await evaluator.evaluate(
    eval_input=EvalInput(
        actual_output="Paris",
        expected_output="Paris",
        input="What is the capital of France?"
    ),
    metrics=[
        MetricConfig(metric_type="correctness", threshold=0.7)
    ]
)

print(result.passed)  # True
print(result.metric_results[0].score)  # 0.95
```
