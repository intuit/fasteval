# Decorators API Reference

## Metric Decorators

### @fe.correctness(threshold)
Evaluates semantic correctness against an expected output.

### @fe.relevance(threshold)
Evaluates relevance to the input query.

### @fe.hallucination(threshold)
Detects hallucinated content.

### @fe.toxicity(threshold)
Detects toxic content.

### @fe.bias(threshold)
Detects biased content.

### @fe.criteria(criteria_text, threshold)
Evaluates against custom criteria written in plain English.

**Aliases:** `@fe.geval()`, `@fe.g_eval()` (backward-compatible)

## Deterministic Decorators

### @fe.contains(case_sensitive)
Checks substring containment.

### @fe.exact_match(normalize, case_sensitive)
Checks exact equality.

### @fe.rouge(threshold, rouge_type)
Computes ROUGE scores.

### @fe.json_schema(schema)
Validates against a JSON schema.

## Data Decorators

### @fe.csv(filepath)
Loads test cases from a CSV file.

### @fe.conversation(turns)
Defines conversation-based tests.

### @fe.traces()
Evaluates production traces. See [Production Traces](/docs/advanced/traces).

## Custom Metric

### @fe.metric(metric_instance)
Applies a custom metric instance. See [Custom Metrics](/docs/advanced/custom-metrics).
